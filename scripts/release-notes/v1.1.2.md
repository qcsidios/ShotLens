## 更新内容

- 提升复杂网页截图的翻译成功率，修复部分文本反复显示“翻译失败”的问题。
- 翻译请求改用更稳定的 JSON 文本数组格式，减少序号混入译文或段落数量错位。
- 模型漏译少量段落时只补译失败内容，已成功的译文保持不变，兼顾成功率与速度。
- 改善产品名、型号、长技术句、标点句、希腊字母和上标混排内容的判断。
- “重新翻译”继续复用当前截图重新 OCR，不会重复截图或要求重新框选。

## 兼容性

- 支持 macOS 14.0 及以上版本。
- 用户 API 设置、快捷键和系统权限身份保持不变。

## 已知限制

- 默认福利 Key 可能受服务商限额或策略影响，重度用户建议填写自己的 API Key。
- 本版本继续使用本地自签名，不代表 Apple 认证；首次启动仍可能出现 Gatekeeper 提示。
