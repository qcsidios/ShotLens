## 更新内容

- 框选后只处理完整英文文字；中文、边缘残字、图标和符号不再进入翻译，中文之间的英文会分别翻译并原位呈现。
- 提升浅色、低对比度文字的 OCR 识别率，并缩短不必要的语言检测流程。
- 译文按原位置采样背景和文字颜色覆盖原文，不再统一显示白底卡片。
- 翻译链路改为一次结构化请求，常见 UI 词优先本地翻译；当前会话内重复内容可直接复用结果。
- 安装在“应用程序”目录时，启动发现新版本会自动下载、验证、替换并重启；更新失败时保留或恢复当前可用版本。

## 兼容性

- 支持 macOS 14.0 及以上版本。
- 用户 API 设置、快捷键、屏幕录制权限身份和现有使用习惯保持不变。

## 已知限制

- 默认福利 Key 可能受服务商限额或策略影响，重度用户建议填写自己的 API Key。
- 本版本继续使用本地自签名，不代表 Apple 认证；首次启动仍可能出现 Gatekeeper 提示。
