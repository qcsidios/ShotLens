## 更新内容

- 修复译文在 Retina 与非整数缩放场景下的位置漂移，严格按 OCR 原坐标锚定。
- 移除译文背后的纯色底板；仅清除原文字形，并使用周边背景像素重建原位置，尽量保留渐变与纹理。
- 译文改用系统中文字体并贴合原行高，减少基线和字号造成的视觉偏移。

## 兼容性

- 支持 macOS 14.0 及以上版本。
- 用户 API 设置、快捷键、屏幕录制权限身份和现有使用习惯保持不变。

## 已知限制

- 默认福利 Key 可能受服务商限额或策略影响，重度用户建议填写自己的 API Key。
- 本版本继续使用本地自签名，不代表 Apple 认证；首次启动仍可能出现 Gatekeeper 提示。
