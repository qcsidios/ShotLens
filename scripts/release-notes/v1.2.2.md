## 更新内容

- 修复点击对勾后部分应用优先粘贴译文文本的问题；现在对勾只复制渲染后的 PNG 截图。
- 修复 OCR 文字颜色估计偏差时，原文字形可能残留为竖条、黑块或色块乱码的问题。
- 加固背景重建图像的像素内存管理，避免偶发重绘异常。

## 兼容性

- 支持 macOS 14.0 及以上版本。
- 用户 API 设置、默认限免模型、快捷键、屏幕录制权限身份和现有使用习惯保持不变。

## 已知限制

- 默认福利 Key 可能受服务商限额或策略影响，重度用户建议填写自己的 API Key。
- 本版本继续使用本地自签名，不代表 Apple 认证；首次启动仍可能出现 Gatekeeper 提示。
