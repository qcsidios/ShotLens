## 更新内容

- 重构截图翻译主链路：OCR 英文片段直接按稳定编号批量翻译，移除段落与文章类型猜测，减少不必要处理和坐标漂移。
- 中文、图标、符号及框选边缘的不完整文字保持原样，只翻译完整英文片段；中英文混排时译文仍嵌回原位置。
- 开启 Vision 英文语言纠错并加强浅色文字识别。实测可修正小型浅色标签误识别，并减少渲染残影被当成英文乱码。
- 单个英文片段漏译或格式异常时保留其他可靠译文，不再让局部问题导致整次翻译失败。
- 常见界面词和同会话精确结果优先使用本地翻译；重新翻译会复用 OCR 结果，只重新调用翻译接口。
- 翻译接口改为单次编号映射，并在本地兼容常见近似 JSON，不再追加网络修复请求；请求超时保持有界。
- 背景重建失败时只在原文字的精确范围内使用采样背景兜底，避免扩大色块或译文卡片。

## 兼容性

- 支持 macOS 14.0 及以上版本。
- 用户 API 设置、默认限免模型、快捷键、屏幕录制权限身份和现有使用习惯保持不变。
- 安装在“应用程序”目录中的旧版本可通过应用内自动更新升级。

## 已知限制

- 翻译速度仍会受网络和模型服务商响应影响；默认请求会在超时后快速失败，避免长时间卡住。
- 默认福利 Key 可能受服务商限额或策略影响，重度用户建议填写自己的 API Key。
- 本版本继续使用本地自签名，不代表 Apple 认证；首次启动仍可能出现 Gatekeeper 提示。
