## 更新内容

- OCR 改为原图优先、增强图补回浅色漏字，保留中英文、数字和标点的完整上下文，并排除纯图标、纯符号和框选边缘的不完整文字。
- 按行距、对齐和视觉样式恢复完整标题、段落和卡片语义，不再逐词孤立翻译；多栏、徽标和图例保持独立。
- 翻译协议由强制 JSON 改为简单编号文本行，同时兼容编号对象、数组和不完整 JSON，单项异常不再导致整批失败。
- 中英文混排时只翻译英文，并校验已有中文没有被模型删除；DeepSeek 官方接口使用非思考模式以降低延迟。
- 原文字区域改为整体背景重建后再绘制译文，不再添加白底或色块，并修复相邻文字污染造成的竖条与乱码。

## 兼容性

- 支持 macOS 14.0 及以上版本。
- 用户 API 设置、默认限免模型、快捷键、屏幕录制权限身份和自动更新方式保持不变。
- 安装在“应用程序”目录中的旧版本可通过应用内自动更新升级。

## 已知限制

- 翻译速度仍取决于网络和模型服务商；实测默认限免模型明显慢于 DeepSeek 等低延迟接口。
- 默认福利 Key 可能受服务商限额或策略影响，重度用户建议填写自己的 API Key。
- 本版本继续使用本地自签名，不代表 Apple 认证；首次启动仍可能出现 Gatekeeper 提示。
